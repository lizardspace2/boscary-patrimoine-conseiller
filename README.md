# Boscary Patrimoine

## Project info

Cabinet de conseil en gestion de patrimoine.

## Development

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

### Setup

```sh
# Install dependencies
npm i

# Start development server
npm run dev
```

### Build

```sh
# Build for production
npm run build
```
